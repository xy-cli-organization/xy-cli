## Project initialization
### Install vue-cli
```bash
npm install -g @vue/cli
```

### Install xy-vue-cli
```bash
npm install -g xy-vue-cli
```

### Init project
```bash
xy-vue-cli init
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
